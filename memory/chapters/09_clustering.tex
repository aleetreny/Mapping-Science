\section{From Static Shape to Dynamic Typologies}

The previous chapters described scientific morphology as a set of continuous relations: full-period shape differences across domains, temporal movement across five publication windows, and pairwise similarity between fields and subfields. This chapter uses clustering to compress those relations into a small typological vocabulary. Clustering is used here as a compression device, not as a discovery of natural kinds. The question is not whether science really has a fixed number of morphological categories, but whether repeated profile patterns can be named without hiding the continuity of the space.

Three axes are kept distinct. Static morphological shape clusters the 241 analysis subfields by their eleven robust-scaled full-period morphology metrics: centroid dispersion, local density and hubness, spectral structure, centroid drift, radial expansion, and recent novelty. Dynamic morphological change clusters subfields by how their windowed structural metrics move over time. Semantic-location clustering, reported mainly in Appendix~C, clusters subfield centroid embeddings and hybrid centroid--morphology representations as sensitivity checks. These three representations answer different questions. Static shape asks what a subfield's full-period morphology looks like; temporal trajectories ask how that morphology changes; semantic location asks where the subfield sits in embedding space.

The clustering inputs remain metric profiles or trajectory features derived from the original SPECTER2 space. Neither OpenAlex labels nor projection coordinates are used to assign clusters. PCA, PCoA, and UMAP appear below only as visual summaries; none of them is used to assign clusters. This distinction matters because a visually tidy projection can be persuasive without being the clustering geometry.

\section{Static Morphological Typologies}

The static typology preserves the solution introduced for the eleven-metric morphology profile: Ward hierarchical clustering, Euclidean distance, robust median/IQR scaling, and \(k=5\). The selected solution has silhouette \(=0.133\). That value is deliberately not treated as strong separation. It says that full-period morphology contains useful anchors, but also that the 241 subfields occupy fuzzy profile regions rather than clean islands.

The five static groups are named by their mean morphology profiles, not by domain composition. T1 Broad sparse profiles is the largest group, with high nearest-neighbor and centroid distances but comparatively low local unevenness. T2 Compact low-dispersion profiles is compact in distance terms while retaining comparatively high dimensionality and hub concentration. Static morphology provides descriptive anchors for the empirical chapters, but the geometry is continuous.

\section{Temporal-Trajectory Typologies}

The dynamic typology asks a different question: whether subfields are easier to group by how their morphology changes than by their full-period shape. The unit is the set of subfields with complete five-window structural trajectories, giving 240 complete cases. Each subfield is represented by the linear slopes of the eight windowed structural metrics: three centroid-dispersion measures, three local density and hubness measures, and two spectral measures. The selected solution uses average-link hierarchical clustering with correlation distance at \(k=4\). It has silhouette \(=0.396\), mean subsample ARI \(=0.725\), AMI \(=0.048\) against the static morphology typology, and AMI \(=0.093\) against OpenAlex domains.

\input{tables/tab_09_dynamic_typology_summary}

These diagnostics justify moving the temporal result into the main chapter. Dynamic morphology produces a clearer and more stable typology than static morphology, but the low agreement with both static typologies and OpenAlex domains shows that it is not simply recovering either full-period shape or taxonomy. The four groups describe movement.

D1 Compacting locally uneven trajectories contains 93 subfields. Its average slope profile shows falling centroid and nearest-neighbor distances, alongside increasing kNN variability and hub concentration. D2 Broadening hub-diffusing trajectories contains 58 subfields and moves in almost the opposite direction on several dimensions: centroid and kNN distances rise, while hubness and spectral complexity fall. D3 Smoothing sparse-neighborhood trajectories contains 54 subfields; nearest-neighbor distance rises, but local unevenness and hubness decline while spectral spread increases. D4 Dimensionalizing hub-concentrating trajectories is smaller, with 35 subfields; PCA dimensionality and hubness rise while outer centroid spread and local variability fall.

The temporal clustering therefore strengthens the chapter without turning it into a stronger static taxonomy. Subfields are easier to group by direction of change than by full-period morphology, yet those dynamic groups remain exploratory. They are best read as trajectory types: compacting, broadening, smoothing, and dimensionalizing tendencies in the morphology of semantic space.

\section{Static, Dynamic, and Semantic-Location Structure Compared}

The static and dynamic typologies organize different aspects of the same subfield universe. To check whether the visual contrast depends on one projection method, Figure~\ref{fig:static_dynamic_projection_comparison} projects both spaces with principal component analysis (PCA), principal coordinates analysis (PCoA), and UMAP. The static panels use robust-scaled eleven-metric profiles; the dynamic panels use standardized temporal slope vectors. PCoA uses the distance geometry closest to the selected clustering in each case: Euclidean distance for static morphology and correlation distance for temporal trajectories.

The bottom row projects standardized temporal slope vectors and colors the selected average-link \(k=4\) dynamic typology. PCA, PCoA, and UMAP are visualization methods only; clustering assignments come from the static Ward and dynamic average-correlation solutions, not from the plotted coordinates.}
\label{fig:static_dynamic_projection_comparison}
\includegraphics[width=\textwidth]{figures/fig_09_static_dynamic_projection_comparison.png}

The contrast is stable enough to be useful but not categorical. In the static panels, the typologies remain visibly continuous and overlapping under both PCA and PCoA; UMAP separates some local neighborhoods more strongly, so it is read cautiously as a neighborhood-preserving view rather than proof of discrete classes. The dynamic panels show clearer regional structure under PCA and especially under correlation-based PCoA, with UMAP giving a sharper but still visualization-only version of the same tendency. This pattern is consistent with the diagnostics in Figure~\ref{fig:static_dynamic_diagnostics}: temporal trajectories are easier to summarize typologically than full-period static shape, but neither projection defines the clusters.

Domain composition gives a supporting caution rather than a main visual result, so the detailed stacked-bar diagnostic is moved to Appendix~C. Static typologies have visible domain tendencies: for example, T1 and T4 are strongly Physical Sciences, while T3 has more Health Sciences and Social Sciences. Dynamic typologies are also not domain-free, but they are less taxonomy-driven than a topic-location partition would be. D1 is mixed between Health Sciences and Physical Sciences, D2 is mostly Physical Sciences with Social Sciences present, D3 is mostly Social Sciences, and D4 is distributed across Life Sciences, Physical Sciences, Health Sciences, and Social Sciences.

The semantic-location sensitivity checks in Appendix~C complete the comparison. Clustering subfield centroid embeddings separates somewhat more clearly than static morphology, with silhouette \(=0.236\) for the selected balanced \(k=3\) centroid solution. It also aligns more strongly with OpenAlex domains (AMI \(=0.471\)) than with the morphology typology (AMI \(=0.084\)). A Ward \(k=5\) centroid variant aligns even more strongly with domains (AMI \(=0.569\)) while still agreeing weakly with morphology (AMI \(=0.095\)). The hybrid centroid--morphology check is stable, with silhouette \(=0.247\), cluster sizes \(91/77/73\), and mean subsample ARI \(=0.951\), but that stability comes with domain-like structure: AMI is \(0.591\) against OpenAlex domains, \(0.505\) against centroid clusters, and only \(0.092\) against morphology.

This comparison sharpens the interpretation. Semantic location is more domain-aligned and somewhat more separable, but it answers a different question from morphology. Hybrid clustering becomes stable largely because it reintroduces semantic-location structure. The temporal result is important because it remains morphology-based while describing a clearer axis of organization: change through time rather than full-period shape.

\section{Interpretive Limits of Typological Clustering}

The evidence supports typological language, not categorical claims. Static morphology is weakly clustered; temporal movement is more clearly typological; semantic location is more domain-aligned. None of these statements implies objective disciplinary borders, quality rankings, causal mechanisms, or replacements for the OpenAlex hierarchy. The typologies are descriptive labels for repeated combinations of metrics.

This is also why the chapter preserves the weak static result instead of replacing it with the stronger temporal or semantic-location alternatives. The static Ward \(k=5\) typology summarizes the full-period shape vocabulary used across Chapters~6--8. The temporal typology adds a second morphology-based vocabulary for trajectories. The centroid and hybrid checks define the boundary of the claim: once semantic location receives substantial weight, clustering becomes more domain-like and no longer describes morphology alone.

The result is therefore not that science has fixed morphological categories. It is that static shape, temporal change, and semantic location provide distinct readings of scientific structure. Static profiles offer fuzzy anchors, temporal trajectories reveal clearer dynamic tendencies, and centroid embeddings recover more topic-location structure. Chapter~9 closes the empirical arc by keeping those axes separate: a map legend for morphology, not a new taxonomy of science.
