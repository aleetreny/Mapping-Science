\section{Deprecated Draft: Embedding-Space Morphological Metrics}

This chapter draft is not included by \texttt{memory/memoria.tex}. The active
thesis specification uses eight structural morphology metrics: centroid-distance
median, centroid-distance IQR, centroid-distance P90, median kNN distance, kNN
distance coefficient of variation, kNN in-degree Gini, PCA \(D_{80}\), and PCA
spectral entropy.

Centroid drift is retained separately as net semantic displacement between the
2000--2004 and 2020--2024 centroid positions. It is not part of the structural
profile matrices used for PCA, clustering, morphology distances, nearest
neighbors, convergence, divergence, or dynamic profile displacement.

The active formula reference is Appendix~\ref{app:metric-formulations}.
