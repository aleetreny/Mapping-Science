# Legacy Exploration Archive

This folder preserves work that is no longer part of the active thesis
pipeline. It includes UMAP-derived morphology metrics as main quantitative
evidence, UMAP-vs-embedding comparisons, metric-space clustering,
dimensionality-reduction shootouts, semantic-distance experiments, and older
diagnostic scripts/tests/docs.

The active repository uses SPECTER2 embedding-space metrics, especially the
reduced 11-metric core. Files here are kept for audit/history only and should
not be used as active pipeline commands without first moving them back into the
main project and updating the documentation.
