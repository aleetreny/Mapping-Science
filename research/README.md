# Thesis Context

## Current Thesis Question

How can the morphology and evolution of scientific disciplines be compared
using structural metrics measured directly in SPECTER2 embedding space?

## Scope

- Unit of analysis: OpenAlex subfields.
- Aggregation levels: fields and domains.
- Analysis period: 2000-2024.
- Text source: title plus abstract.
- Embedding model: SPECTER2.
- Main metric set: reduced 11-metric embedding-space core.
- UMAP role: visualization only.

## Out Of Scope

The active thesis does not include growth prediction, regression targets,
classification, dashboards, UMAP-derived metrics as main evidence, or mixed
UMAP-plus-embedding clustering.

Historical explorations are preserved in `archive/legacy_exploration/` for
auditability.
