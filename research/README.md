# Research workspace

This folder is the working knowledge base for the TFM.

## Purpose

It is meant to centralize:

1. project framing
2. literature review notes
3. competing tools and prior projects
4. methodological options
5. open questions
6. possible experiments
7. data source comparisons
8. idea bank for future agents

## Current files

1. `KB_master.md`
   Main strategic context and project framing.

2. `ideas_database.md`
   Structured bank of ideas, experiments, and hypotheses.

## Intended workflow

1. keep high level decisions in `KB_master.md`
2. keep exploratory ideas and literature inspired directions in `ideas_database.md`
3. later add more files for:
   1. formal literature review
   2. corpus decisions
   3. experiment logs
   4. visualization design
   5. predictive modeling notes

## Note

The goal is not to dump random notes, but to build a reusable context base so future agents can continue the project without losing the thread.
