# TFM Summary

This TFM compares the morphology and evolution of scientific disciplines using
OpenAlex title-plus-abstract text embedded with SPECTER2.

The current thesis is descriptive and comparative. It does not train growth
prediction models, classifiers, dashboards, or UMAP-derived metric systems for
main evidence.

The core methodological choice is to measure discipline structure in the
original SPECTER2 embedding space. The final downstream feature set is the
eight-metric structural core documented in [reduced_metric_core.md](reduced_metric_core.md).

Main analyses:

- Static comparison of subfields, fields, and domains.
- Temporal evolution across five 5-year windows from 2000 to 2024.
- Morphological convergence and divergence between disciplines.
- Auxiliary UMAP visualizations for inspection only.
